Basics of HTML

Introduction to HTML

HTML (Hypertext Markup Language) is the standard language used to create web pages. It provides the structure for web content.

Structure of an HTML Document

An HTML document is made up of various elements. The basic structure includes:

<html> <head> <body>

HTML Tags and Attributes

HTML tags define elements, and attributes provide additional information about the elements. For example, the <img> tag has the src attribute to specify the image source:

<img src="image.jpg" alt="Image">

Basic Tags

The basic tags that structure an HTML document include:

  • <html>: Defines the HTML document
  • <head>: Contains metadata (like title, character set, etc.)
  • <body>: Contains the visible content of the webpage
  • <title>: Sets the title of the page
  • <meta>: Provides metadata such as character set and viewport settings

HTML Doctype Declaration

The <!DOCTYPE html> declaration defines the document type and version of HTML. It should be the first line in any HTML document to specify that the document is HTML5:

<!DOCTYPE html>

HTML Comments

HTML comments are written within <!-- comment --> tags. These comments are not displayed on the page, but can help with documentation:

<!-- This is a comment -->

<code> tag:

The <code> tag is used to define a block of code or an inline code fragment on a webpage. The text inside it is typically rendered in a monospace font, which makes it stand out from regular text. This is helpful for displaying code snippets in documentation or tutorials.

< and >:

< and > are HTML character entities that represent the less-than ( <) and greater-than (>) signs, respectively. Since < and> are special characters in HTML (used for tags), you need to use these entities to display them as literal characters on the page.

Without these entities, the browser would interpret < and> as the start and end of an HTML tag, which could break the rendering of the code.
